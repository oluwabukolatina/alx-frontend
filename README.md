## Installations

#### Getting started

#### Clone

- Clone this project to your local machine `https://github.com/oluwabukolatina/alx-frontend .git`

#### Setup

- Installing the project dependencies
  > Run the command below
  ```shell
  $ npm install
  ```
- Start your node server
  > run the command below
  ```shell
  $ npm start
  ```
- Use `http://localhost:8080`
